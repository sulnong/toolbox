## Why

创建一个合规安全的魔兽世界游戏助手，专注于辅助功能而非全自动机器人，帮助玩家提升游戏体验的同时避免违反用户条款导致的封号风险。

## What Changes

- 添加 `wow-game-agent` 核心能力：智能游戏辅助系统
- 实现人性化输入模拟：贝塞尔曲线鼠标轨迹、动态延迟
- 创建安全框架：防检测机制、行为随机化、实时监控
- 开发钓鱼场景插件：合规的钓鱼辅助工具
- 建立场景插件系统：可扩展的辅助功能框架

## Impact

- 受影响规范：无 (新增能力)
- 受影响代码：
  - 新增 `wow-game-agent/` 目录
  - 核心安全库：human-input, behavior-randomizer, safety-monitor
  - 场景插件：fishing-assistant, 可扩展接口
- 外部依赖：增加 Windows API 相关库、AI 模型集成
- 安全考虑：所有操作需要用户确认，不提供无人值守功能

## 合规性重点

**严格遵守魔兽世界用户条款**：
- 不实现全自动游戏操作
- 所有关键操作需要用户明确触发
- 模拟真实人类行为模式
- 提供实时监控和紧急停止机制
- 不修改游戏客户端或内存

**防封策略**：
- 人性化输入模拟
- 行为随机化
- 使用量限制
- 透明化操作日志
